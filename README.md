# LAN
## Popis sítě
- Rozsah: 192.168.176.0/24
- S1: přípojné místo pro PC1, PC2 a Laptop
- S2: přípojné místo pro SRV1, SRV2
- SRV1: DHCP + DNS (hodbod.cz)
- SRV2: WEB server
- PC1: statická konfigurace
- PC2: dynamická konfigurace přes DHCP

## Výpočet X
H(72)+O(79)+D(68)+B(66)+O(79)+D(68) = 432
432 mod 256 = **176**
